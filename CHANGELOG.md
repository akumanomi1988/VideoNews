# Changelog

All notable changes to VideoNews will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Comprehensive documentation in `/docs` directory
- Contribution guidelines
- Changelog tracking

### Changed
- Improved README with architecture diagrams
- Enhanced error handling in pipeline stages
- Optimized memory management in video processing

## [1.0.0] - 2025-05-10

### Added
- Initial release
- Core video processing pipeline
- Support for short-form and long-form video generation
- Multi-provider TTS support (Edge, ElevenLabs, Bark)
- AI-powered content generation and enhancement
- Telegram bot integration
- Monitoring dashboard
- SQLite database for tracking processed articles
- User management system

### Security
- API key management system
- Rate limiting implementation
- OAuth2 integration for social platforms

## [0.9.0] - 2025-04-15

### Added
- Beta release
- Basic pipeline functionality
- News article processing
- Video generation capabilities
- Platform upload support

### Fixed
- Memory leaks in video processing
- Audio synchronization issues
- Subtitle timing problems

## [0.8.0] - 2025-03-01

### Added
- Alpha release
- Core architecture implementation
- Basic news processing
- Preliminary AI integration

### Known Issues
- Memory management in video processing
- Audio-subtitle synchronization
- Resource cleanup in error cases
